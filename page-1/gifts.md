# Gifts

## Commands and Mechanics

Run the `/gift` command to view your gifts and check if you can claim your free gift. Users can get free gifts by clicking on the claim button every **12** hours.

## Small Gift

| Reward       | Range           |
| ------------ | --------------- |
| **Money**    | **$100 - $200** |
| **Logs**     | **7 - 16**      |
| **Saplings** | **1 - 2**       |
| **Cutters ** | **0 - 1**       |
| **Plots**    | **0 - 1**       |
| **Vehicles** | **0 - 1**       |
